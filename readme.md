# What's this about?
Do you think the universe was born from randomness or from a grand design?

This interactive webpage takes you on a thought-provoking journey through these questions. As you scroll through the page, you'll be met with a staggering 100,000 characters of pure randomnessâ€”your task is to find any word that holds meaning.

# Run Locally

To run the app locally:

Install dependencies:
`npm install`

Run it:
`npm run start`

To change the length of the randomly generated text, change the number in the `textLength` variable in the `index.js` file
